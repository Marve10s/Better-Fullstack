title: React Native 选项
description: Expo 和 React Native 中 frontend、navigation、UI、storage、testing、push、OTA 和 links 的选项值。
---

React Native 选项通过 `--ecosystem react-native` 选择。它们会生成 Expo app，并且不暴露 TypeScript Web frontend、backend、database 或 Web deployment 分类。

<Callout kind="info" title="兼容性说明">
`native-uniwind` 会选择 `mobile-ui: uniwind`。`native-unistyles` 会选择 `mobile-ui: unistyles`。Tamagui 和 gluestack-ui 面向 `native-bare`，以避免样式系统冲突。
</Callout>
