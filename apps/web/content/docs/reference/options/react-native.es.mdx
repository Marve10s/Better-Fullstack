Las opciones de React Native se seleccionan con `--ecosystem react-native`. Generan una app Expo y no exponen categorías de frontend web TypeScript, backend, base de datos ni deploy web.

<Callout kind="info" title="Notas de compatibilidad">
`native-uniwind` selecciona `mobile-ui: uniwind`. `native-unistyles` selecciona `mobile-ui: unistyles`. Tamagui y gluestack-ui estan pensados para `native-bare` para evitar sistemas de estilos en conflicto.
</Callout>
