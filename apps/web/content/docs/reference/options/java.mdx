Categories applied when scaffolding a Java project.

This page covers **17 options** across **6 categories**.

Use CLI values in commands and MCP payloads.

<Callout kind="info" title="Compatibility notes">
Spring Boot requires Maven or Gradle. Java ORM, auth, and Java libraries require Spring Boot plus a build tool. Flyway requires Spring Data JPA. `springdoc-openapi` is not currently a supported Java library option.
</Callout>
