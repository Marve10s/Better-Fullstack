TypeScript is the broadest Better Fullstack ecosystem. This page keeps the TypeScript option surface in one place, matching the single-page pattern used by Rust, Python, Go, and Java.

Use CLI values in commands and MCP payloads.

<Callout kind="info" title="Compatibility notes">
The tables list valid option values, not every valid combination. tRPC is React-oriented. `backend: self` is for fullstack frameworks and pairs with `runtime: none`. Qwik, Angular, Redwood, and Fresh have narrower API/backend support. Workers runtime, PWA, UI libraries, payments, and file storage all have stack-specific constraints.
</Callout>
