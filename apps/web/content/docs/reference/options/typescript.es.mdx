TypeScript es el ecosistema más amplio de Better Fullstack. Usa estos valores de CLI en comandos, payloads MCP y enlaces del Stack Builder.

<Callout kind="info" title="Notas de compatibilidad">
La tabla lista valores validos, no todas las combinaciones validas. tRPC está orientado a React. `backend: self` es para frameworks fullstack y se combina con `runtime: none`. Workers runtime, librerias UI, providers, addons y destinos de deploy tienen restricciones específicas de stack.
</Callout>

Las opciones de React Native y Expo viven en la pagina [Opciones de React Native](/docs/reference/options/react-native/).
