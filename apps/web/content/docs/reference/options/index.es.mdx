Esta referencia lista los valores de CLI aceptados por Better Fullstack. Usa estos valores en comandos, URLs del Stack Builder y payloads MCP.

Algunas etiquetas del builder se normalizan al mismo valor de CLI, y no todas las opciones son validas con cada frontend, backend, runtime o ecosistema. Ejecuta una verificacion de compatibilidad antes de crear scaffolds de stacks complejos.

<OptionReferenceSummary />

<Callout kind="info" title="La compatibilidad importa">
Las tablas listan valores de opcion validos. El compatibility checker aun decide si una combinacion completa de stack es valida y puede rechazar o ajustar combinaciones antes de generar.
</Callout>
