这份参考列出 Better Fullstack 接受的 CLI 值。请在命令、Stack Builder URLs 和 MCP payloads 中使用这些值。

一些 builder labels 会规范化为相同的 CLI 值，而且并非每个选项都能与每个 frontend、backend、runtime 或 ecosystem 搭配。在 scaffold 复杂 stack 之前，请运行兼容性检查。
