Java 项目支持 Spring Boot、Quarkus 和普通 Java 模板。为构建工具、持久化、认证、应用库和测试库选择 Java 选项。

## 前置要求

- Java 21。
- 如果要初始化仓库，需要 Git。

## 脚本化示例

```npm
npm create better-fullstack@latest my-java-api -- \
  --ecosystem java \
  --java-web-framework spring-boot \
  --java-build-tool maven \
  --java-orm spring-data-jpa \
  --java-auth spring-security \
  --java-api spring-graphql \
  --java-logging logback \
  --java-libraries spring-actuator spring-validation flyway \
  --java-testing-libraries junit5 mockito testcontainers \
  --ai-docs none \
  --version-channel stable \
  --no-install \
  --no-git
```

## Java 分类

| 分类 | 值 |
| --- | --- |
| Web 框架 | `spring-boot` `quarkus` `none` |
| 构建工具 | `maven` `gradle` `none` |
| ORM | `spring-data-jpa` `jooq` `mybatis` `none` |
| Auth | `spring-security` `keycloak` `none` |
| API | `spring-graphql` `none` |
| 日志 | `logback` `none` |
| 库 | `spring-actuator` `spring-validation` `flyway` `liquibase` `springdoc-openapi` `lombok` `mapstruct` `caffeine` `resilience4j` `spring-webflux` `spring-batch` `spring-kafka` `spring-mail` `spring-devtools` `micrometer-prometheus` `thymeleaf` `spring-amqp` `opentelemetry-java` `none` |
| 测试库 | `junit5` `mockito` `testcontainers` `assertj` `rest-assured` `wiremock` `awaitility` `archunit` `jqwik` `none` |

## 兼容性说明

- Spring Boot 和 Quarkus 需要 Maven 或 Gradle。
- Java ORM、auth 和 Spring 专用库需要 Spring Boot 加 Maven 或 Gradle。
- Flyway 需要 Spring Data JPA。
- 一些库是框架专用的。例如 Spring Batch、Spring Kafka、Spring Mail、Spring WebFlux、Springdoc OpenAPI、Actuator 和 Spring Security 面向 Spring Boot 项目。

## 生成行为

- 启用安装时，Maven 项目使用 `./mvnw test` 验证。
- 启用安装时，Gradle 项目使用 `./gradlew test` 验证。
- 通过使用 `--java-web-framework none`，并为依赖框架的分类选择兼容的 `none` 值，可以选择普通 Java 行为。
- AI 指令文件仍可通过 `--ai-docs` 使用，包括 `--ai-docs none`。
