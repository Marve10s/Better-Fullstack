React Native es el ecosistema mobile dedicado. Genera proyectos Expo y mantiene las opciones mobile separadas de categorías TypeScript web frontend, backend, database y deploy.

## Prerrequisitos

- Node.js 20+.
- npm, pnpm, Yarn o Bun.
- Tooling iOS/Android compatible con Expo cuando quieras validar en simulador o dispositivo.

## Ejemplo scriptado

```npm
npm create better-fullstack@latest mobile-app -- \
  --ecosystem react-native \
  --frontend native-bare \
  --mobile-navigation expo-router \
  --mobile-ui tamagui \
  --mobile-storage mmkv \
  --mobile-testing maestro-react-native-testing-library \
  --mobile-push expo-notifications \
  --mobile-ota expo-updates \
  --mobile-deep-linking expo-linking \
  --ai-docs none \
  --no-install
```

## Categorias de scaffold mobile

| Categoria | Valores |
| --- | --- |
| Native frontend | `native-bare` `native-uniwind` `native-unistyles` |
| Navigation | `expo-router` `react-navigation` `none` |
| UI | `tamagui` `gluestack-ui` `uniwind` `unistyles` `none` |
| Storage | `mmkv` `none` |
| Testing | `maestro` `react-native-testing-library` `maestro-react-native-testing-library` `none` |
| Push | `expo-notifications` `none` |
| OTA | `expo-updates` `none` |
| Deep linking | `expo-linking` `none` |

## Integraciones generadas

- Config Expo con scheme/runtime settings cuando se selecciona deep linking u OTA.
- Archivos de rutas Expo Router o un entrypoint React Navigation, segun navigation.
- Provider/theme mobile UI para Tamagui, Gluestack UI, Uniwind o Unistyles.
- Helpers cliente MMKV cuando se selecciona storage.
- Ejemplos Jest Expo y React Native Testing Library, flujos Maestro o ambos.
- Helpers de registro push para Expo Notifications.
- Helpers de OTA update para Expo Updates.
- `.env.example` con convenciones `EXPO_PUBLIC_` para configuración de cliente mobile.

## Notas de compatibilidad

- React Native no expone categorías TypeScript web frontend, backend, database, web UI, web deploy ni server deploy.
- Los modos UI Uniwind y Unistyles estan ligados a sus variantes Expo frontend correspondientes.
- Tamagui y Gluestack UI usan la variante bare Expo para evitar conflictos de styling.
- Deep linking se activa automaticamente cuando un flujo mobile auth necesita ejemplos de redirect URI.
