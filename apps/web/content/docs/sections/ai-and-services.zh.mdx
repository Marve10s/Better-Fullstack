服务选择会围绕核心应用添加 provider clients、环境变量、示例或集成脚手架。

## AI

TypeScript AI 选择包括 Vercel AI SDK、Mastra、VoltAgent、LangGraph、OpenAI Agents SDK、Google ADK、ModelFusion、LangChain、LlamaIndex、TanStack AI、AI CLI 或 none。

Python 有单独的多选 AI 分类，包含 LangChain、LlamaIndex、OpenAI SDK、Anthropic SDK、LangGraph、CrewAI、Haystack 或 none。

## 产品服务

常见服务分区包括：

- Email：React Email、Resend、Nodemailer、Postmark、SendGrid、AWS SES、Mailgun、Plunk 或 none。
- Realtime：Socket.IO、PartyKit、Ably、Pusher、Liveblocks、Yjs 或 none。
- Jobs：BullMQ、Trigger.dev、Inngest、Temporal 或 none。
- CMS：Payload、Sanity、Strapi、TinaCMS、Directus 或 none。
- Analytics：Plausible、Umami 或 none。
- Feature flags：GrowthBook、PostHog、LaunchDarkly、Flagsmith、Unleash 或 none。
