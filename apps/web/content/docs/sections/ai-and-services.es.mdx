Las opciones de servicios agregan clientes de provider, variables de entorno, ejemplos o scaffolds de integracion alrededor de la app core.

## IA

Las opciones de IA para TypeScript incluyen Vercel AI SDK, Mastra, VoltAgent, LangGraph, OpenAI Agents SDK, Google ADK, ModelFusion, LangChain, LlamaIndex, TanStack AI, AI CLI o none.

Python tiene una categoría IA separada de seleccion multiple con LangChain, LlamaIndex, OpenAI SDK, Anthropic SDK, LangGraph, CrewAI, Haystack o none.

## Servicios de producto

Las secciones comunes de servicios incluyen:

- Email: React Email, Resend, Nodemailer, Postmark, SendGrid, AWS SES, Mailgun, Plunk o none.
- Realtime: Socket.IO, PartyKit, Ably, Pusher, Liveblocks, Yjs o none.
- Jobs: BullMQ, Trigger.dev, Inngest, Temporal o none.
- CMS: Payload, Sanity, Strapi, TinaCMS, Directus o none.
- Analytics: Plausible, Umami o none.
- Feature flags: GrowthBook, PostHog, LaunchDarkly, Flagsmith, Unleash o none.
