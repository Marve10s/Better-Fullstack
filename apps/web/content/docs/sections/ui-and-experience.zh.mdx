UI 和体验选择会塑造生成的前端代码、设计 primitives 和客户端应用模式。

## Styling 和 UI 库

CSS 选择包括 Tailwind、SCSS、Less、PostCSS-only 或 none。UI 库包括 shadcn/ui、shadcn-svelte、daisyUI、Radix UI、Headless UI、Park UI、Chakra UI、NextUI、Mantine、MUI、Ant Design、Base UI、Ark UI、React Aria 和 none。

shadcn/ui 还有额外 presets，用于基础 primitives、视觉风格、图标、颜色主题、中性色、字体和圆角。

## 表单、验证和状态

前端应用选择包括：

- 状态：Zustand、Jotai、Nanostores、Redux Toolkit、MobX、XState、Valtio、TanStack Store、Legend State 或 none。
- 表单：TanStack Form、React Hook Form、Formik、Final Form、Conform、Modular Forms 或 none。
- 验证：Zod、Valibot、ArkType、TypeBox、Typia、Runtypes、Effect Schema 或 none。

## 体验功能

动画、i18n 和前端相邻的 app-platform addons 在概念上属于这里，即使它们的精确 flags 出现在不同选项表中。

React Native 有自己的移动端 UI 选择：Tamagui、Gluestack UI、Uniwind、Unistyles 或 none。
