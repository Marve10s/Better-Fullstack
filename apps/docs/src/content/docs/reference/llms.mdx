Better Fullstack publishes an `llms.txt` file for AI agents and search tools.

- Site-level file: [`/llms.txt`](https://better-fullstack.dev/llms.txt)
- Docs-specific file: [`/docs/llms.txt`](https://better-fullstack.dev/docs/llms.txt)

The docs-specific file is generated from the same metadata used to build the option reference pages.

## Why it exists

`llms.txt` gives agents a compact, text-first overview of:

- What Better Fullstack does.
- Supported ecosystems.
- Important pages.
- Recommended CLI and MCP workflows.
- Links to dynamic option reference pages.
