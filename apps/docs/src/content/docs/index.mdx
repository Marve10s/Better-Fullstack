Start here when you want a project generated quickly with explicit, reproducible stack choices.

## 1. Install the required tools

Everyone needs Node.js 20 or newer and Git. Bun is the fastest launcher and the default used in this repository.

- [Download Node.js](https://nodejs.org/en/download)
- [Install Bun](https://bun.sh/docs/installation)
- [Download Git](https://git-scm.com/downloads)

```bash
node --version
bun --version
git --version
```

For Rust, Python, Go, or Java projects, install the matching ecosystem toolchain before running generated project commands.

| Ecosystem | Tooling |
| --- | --- |
| TypeScript | Node.js plus Bun, npm, pnpm, or Yarn |
| Rust | Rust toolchain and Cargo |
| Python | Python and uv |
| Go | Go toolchain |
| Java | Java 21; generated Maven or Gradle wrappers handle builds |

## 2. Create a project

Use the interactive flow when exploring choices:

```bash
bun create better-fullstack@latest my-app
```

Use explicit flags when you already know the stack:

```bash
bun create better-fullstack@latest my-app \
  --frontend next \
  --backend self \
  --runtime none \
  --database postgres \
  --orm drizzle \
  --auth better-auth \
  --api trpc
```

## 3. Inspect before installing

For CI and AI-agent workflows, scaffold first and install dependencies after you inspect the generated files.

```bash
bun create better-fullstack@latest my-app --no-install --no-git --ai-docs agents-md
cd my-app
bun install
```

Generated projects include `bts.jsonc`, a reproducible command, framework scripts, and optional AI documentation files.
